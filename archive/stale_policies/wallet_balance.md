# Wallet Balance Mismatch

Use this when customer says balance did not update, money disappeared, or ledger and wallet appear inconsistent.

Required checks:
- Get wallet balance.
- Compare available balance with ledger balance.
- Review recent ledger entries and pending reversals.

Resolution:
- If ledger balance is higher than available balance, create support ticket for wallet reconciliation.
- If pending transaction explains the difference, tell customer the expected settlement window.
