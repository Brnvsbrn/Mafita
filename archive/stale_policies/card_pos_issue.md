# Card And POS Issue

Use this for card issue, virtual card issue, POS terminal complaint, ATM/POS decline, Verve/Visa/Mastercard issue.

Required checks:
- Check card status or POS terminal status.
- For unauthorized card debit, block card if customer confirms.
- For POS settlement issue, check terminal settlement status.

Resolution:
- Create high-priority ticket for duplicate/unauthorized card debits.
- Create agent operations ticket for POS settlement issue.
