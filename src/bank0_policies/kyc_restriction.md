# KYC Restriction

Use this when account is blocked, restricted, limit reached, BVN mismatch, NIN missing, or KYC upgrade is requested.

Required checks:
- Check KYC tier, BVN status, NIN status, and account restrictions.
- If NIN is missing, create KYC update ticket.
- If BVN mismatch exists, create correction ticket and request ID proof.

Resolution:
- Tier 1 customers may hit daily limits quickly.
- BVN/NIN fixes usually take 24-48 hours after valid documents are submitted.
- Do not remove restrictions without verification.
