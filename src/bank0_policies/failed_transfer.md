# Failed Transfer

Use this when a customer says money left their wallet/account but the recipient did not receive it.

Required checks:
- Confirm the customer identity.
- Locate the transaction by reference, Session ID, or recent transaction list.
- Check transaction status and ledger debit.
- If status is failed and debit posted, create a dispute and request reversal.
- If status is successful, give the Session ID and advise recipient bank trace.

Resolution:
- Failed NIP transfers usually reverse within 24-48 business hours.
- If no reversal after 48 hours, escalate to operations.
- Do not promise instant reversal for interbank transfers.
